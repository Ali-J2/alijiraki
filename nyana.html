Nyana

Nyana is a story rich metroidvania set in an oppressive, yet beautifully hand-drawn world.

Nyana on Steam

Development

Nyana is the latest project under development at Orion Games. Its production started as I was working on Bashtronaut, so I did not work on this project initially. Later on, I joined in to create a vertical slice. I was responsible for the physics based grab/throw enemies feature as well as enemy AI and setting up various basic systems for HUD, game feel, environmental interactions, and the intro cutscene.

Platforming

This shows some of the platforming and the grab/throw mechanic featured in Nyana. Nyana uses a dynamic, non-grid based 2D level layout.
