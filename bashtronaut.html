Bashtronaut

Bashtronaut is a hack-and-bash adventure where you play as a seemingly ordinary exterminator in an extraordinary setting.

Bashtronaut on Steam

Development

Bashtronaut is my favorite project I've worked on so far. I was responsible for the combat, game design, and all the mechanics of the game working as the sole programmer on it. That gave me a lot of creative control, and we came out with a really fun, tightly balanced game. Development on this game is complete, but it is scheduled for release at a future date.

Comedic Setting

Bashtronaut has you playing as an employee working for an oppressive corporation that aims to clear different worlds from invaders/aliens.

Frenetic Combat

Using our experience from Arto, we crafted a very fun, tightly balanced combat system that takes place on small arenas.

Crazy Levels

I was responsible for creating many different unique combat scenarios that span across five worlds.

Level Select

This shows how level select works. If the player dies in a level, it's marked so that they can return and retrieve dropped items.

Lots of items

Bashtronaut included a bunch of items that can be used offensively and defensively. Some of them just increase stats, others spawn mines, automated gatling guns, or make you move faster among many other effects.
